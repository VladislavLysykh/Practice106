Описание проекта

Разработано веб-приложение для анализа текста.

Пользователь вводит текст через веб-интерфейс и отправляет его на обработку. Приложение подсчитывает:

количество слов;
количество символов;
наиболее часто встречающееся слово.


Архитектура приложения

Приложение состоит из трёх сервисов:

Frontend

Пользовательский интерфейс, реализованный на HTML и JavaScript.

Backend

Сервис на Flask, принимающий запросы от пользователя и передающий данные на обработку.

Worker

Отдельный сервис для выполнения длительных задач обработки текста.

Nginx

Используется как единая точка входа в приложение.

Сборка и запуск:
